# 🔗 Obsidian 关系图谱工具汇总

## 🎯 已完成的工作

我已经为你创建了一套完整的 Obsidian 关系图谱分析工具，包含以下内容：

## 📊 核心文件

### 1. **数据分析脚本**
- `scripts/create-obsidian-graph.py` - 主分析脚本
  - 分析所有 `.md` 文件的内容
  - 提取关键词和分类
  - 计算文件相似度
  - 生成关系数据

### 2. **分析报告**
- `Obsidian-图谱增强指南.md` - 详细分析报告
  - 文件分类统计
  - 枢纽文件识别
  - 推荐链接建议
  - 使用指南

### 3. **可视化工具**
- `obsidian-graph-visualizer.html` - 交互式可视化
  - 力导向图展示
  - 实时筛选和调整
  - 节点详情查看
  - 响应式设计

### 4. **数据文件**
- `obsidian-graph-data.json` - 图形数据
  - 节点信息（文件）
  - 链接信息（关系）
  - 分类和关键词

### 5. **启动脚本**
- `open-graph-visualizer.sh` - 一键启动
  - 多种打开方式
  - HTTP 服务器
  - 二维码生成

## 🚀 快速开始
