# PDF Split Skill

一个简单易用的PDF拆分工具，可以将大型PDF文件拆分为多个小文件。

## 功能特性

- ✅ 按固定页数拆分（如每30页一个文件）
- ✅ 按指定页面范围提取（如第1-100页）
- ✅ 自动安装依赖（PyPDF2）
- ✅ 自动验证生成的PDF文件
- ✅ 生成文件列表和验证报告
- ✅ 支持中文文件名

## 安装依赖

工具会自动检查并安装PyPDF2。如果需要手动安装：

```bash
pip install PyPDF2
```

## 使用方法

### 1. 按固定页数拆分

```bash
# 默认每30页一个文件
python split_pdf.py "大型文档.pdf"

# 每50页一个文件
python split_pdf.py "大型文档.pdf" 50

# 指定输出目录
python split_pdf.py "大型文档.pdf" 30 --output "我的拆分结果"
```

### 2. 按页面范围提取

```bash
# 提取第1-100页
python split_pdf.py "大型文档.pdf" --range 1-100

# 提取第50-200页
python split_pdf.py "大型文档.pdf" --range 50-200
```

## 输出文件

拆分完成后，会在当前目录下创建输出目录（如`文档名_拆分结果/`），包含：

- `第01部分_第1-30页.pdf` - 拆分后的PDF文件
- `第02部分_第31-60页.pdf` - 拆分后的PDF文件
- ...
- `文件列表.txt` - 所有文件的详细信息（大小、验证状态）

## 文件命名规则

```
第{序号}部分_第{起始页}-{结束页}页.pdf
```

例如：
- `第01部分_第1-30页.pdf`
- `第02部分_第31-60页.pdf`
- `第26部分_第751-752页.pdf`

## 验证功能

工具会自动验证每个生成的PDF文件：
- ✅ 检查文件是否存在
- ✅ 验证PDF格式是否有效
- ✅ 统计实际页数
- ✅ 在文件列表中标记验证状态

## 示例

```bash
# 拆分一个752页的PDF，每30页一个文件
python split_pdf.py "信息系统项目管理师教程（第四版）-带目录可搜索.pdf" 30

# 输出：
# 📄 输入文件: 信息系统项目管理师教程（第四版）-带目录可搜索.pdf
# 📊 每部分页数: 30
# 📁 输出目录: 信息系统项目管理师教程（第四版）-带目录可搜索_拆分结果
# 📖 总页数: 752
# 🔢 将拆分为 26 个文件
# ...
# 🎉 拆分完成!
# ✅ 成功: 26/26 个文件
```

## 注意事项

1. **大文件处理**：处理大型PDF文件可能需要一些时间，请耐心等待
2. **内存使用**：PyPDF2会加载整个PDF到内存，超大文件可能需要更多内存
3. **中文支持**：完全支持中文文件名和路径
4. **文件权限**：确保有读写权限

## 故障排除

### 问题：导入PyPDF2失败
**解决方案**：
```bash
pip install PyPDF2 --upgrade
```

### 问题：文件不存在
**解决方案**：检查文件路径是否正确，使用绝对路径

### 问题：输出目录已存在
**解决方案**：工具会自动创建或使用现有目录
