## 第 11 章 项目成本管理 ⭐⭐⭐⭐⭐ **计算题核心**

### 11.1 规划成本管理

**输出：** 成本管理计划

### 11.2 估算成本

**估算方法：**

| 方法 | 特点 | 准确度 | 适用阶段 |
|------|------|--------|----------|
| 类比估算 | 自上而下，快速 | 低 | 早期 |
| 参数估算 | 基于历史数据 | 中 | 规划期 |
| 自下而上估算 | 逐层汇总，准确 | 高 | 详细规划 |
| 三点估算 | 考虑风险 | 高 | 不确定性强 |

**输出：** 成本估算、估算依据

### 11.3 制定预算

**过程：**
```
活动成本估算 → 汇总 → 应急储备 → 管理储备 → 项目预算
```

**成本基准：** 经批准的时间分段的预算（S 曲线）

**输出：** 成本基准、项目资金需求

### 11.4 控制成本 ⭐⭐⭐⭐⭐ **案例分析必考**

#### 挣值管理 EVM（必背公式）

**三个基本参数：**

| 参数 | 名称 | 含义 | 计算公式 |
|------|------|------|----------|
| PV | 计划价值 | 计划完成工作的预算 | 计划工作量 × 预算单价 |
| EV | 挣值 | 实际完成工作的预算 | 实际工作量 × 预算单价 |
| AC | 实际成本 | 实际完成工作的实际成本 | 实际工作量 × 实际单价 |

**四个评价指标：**

| 指标 | 名称 | 计算公式 | 含义 | 判断标准 |
|------|------|----------|------|----------|
| SV | 进度偏差 | SV = EV - PV | 进度提前/落后 | >0 提前，<0 落后 |
| CV | 成本偏差 | CV = EV - AC | 成本节约/超支 | >0 节约，<0 超支 |
| SPI | 进度绩效指数 | SPI = EV / PV | 进度效率 | >1 提前，<1 落后 |
| CPI | 成本绩效指数 | CPI = EV / AC | 成本效率 | >1 节约，<1 超支 |

**四个预测指标：**

| 指标 | 名称 | 计算公式 | 含义 |
|------|------|----------|------|
| BAC | 完工预算 | 项目总预算 | 原始计划总成本 |
| ETC | 完工尚需估算 | ETC = BAC - EV（典型）<br>ETC = 重新估算（非典型） | 完成剩余工作还需多少成本 |
| EAC | 完工估算 | EAC = AC + ETC<br>或 EAC = BAC / CPI（典型）<br>或 EAC = AC + (BAC - EV)（非典型） | 预计项目总成本 |
| VAC | 完工偏差 | VAC = BAC - EAC | 预计完工时的成本偏差 |

#### 📐 典型偏差 vs 非典型偏差

**典型偏差（当前偏差会持续）：**
```
EAC = BAC / CPI
ETC = EAC - AC
```

**非典型偏差（当前偏差不会持续）：**
```
EAC = AC + (BAC - EV)
ETC = BAC - EV
```

**完工尚需绩效指数 TCPI：**
```
TCPI = (BAC - EV) / (BAC - AC)  按原预算完成
TCPI = (BAC - EV) / (EAC - AC)  按新估算完成
```

**记忆口诀：**
> **挣值减计划是进度偏差（SV=EV-PV）**
> **挣值减实际是成本偏差（CV=EV-AC）**
> **除过去是指数，大于 1 是好的**

## 第 12 章 项目质量管理 ⭐⭐⭐⭐

### 12.1 规划质量管理

**质量 vs 等级：**
- 质量：满足要求的程度（高/低）
- 等级：功能多少（高/低）
- **低质量是问题，低等级不一定是问题**

**质量成本 COQ：**
- **一致性成本**：预防成本 + 评价成本
- **非一致性成本**：内部失败成本 + 外部失败成本

**输出：** 质量管理计划、质量测量指标、核对单

### 12.2 管理质量（质量保证）⭐

**核心：** 审计质量要求和质量控制测量结果

**工具与技术：**
- 质量审计（识别最佳实践、违规情况）
- 过程分析
- 根本原因分析

**输出：** 质量报告、变更请求

### 12.3 控制质量 ⭐⭐⭐⭐

**核心：** 监控并记录质量活动结果

**工具与技术（老 7 种工具）：**

| 工具 | 用途 | 图形 |
|------|------|------|
| 因果图（鱼骨图） | 找根本原因 | 鱼骨状 |
| 流程图 | 展示过程步骤 | 流程图 |
| 核查表 | 收集数据 | 表格 |
| 帕累托图 | 找主要问题（80/20 法则） | 柱状图 + 折线 |
| 直方图 | 展示数据分布 | 柱状图 |
| 控制图 | 判断过程是否稳定 | 折线图 + 控制限 |
| 散点图 | 展示变量关系 | 点阵图 |

**控制图要点：**
- 上控制限 UCL、下控制限 LCL
- 超出控制限或连续 7 点在同侧 → 过程失控

**输出：** 质量控制测量结果、核实的可交付成果

### 质量管理的三个过程对比

| 过程 | 执行者 | 目的 | 时间 |
|------|--------|------|------|
| 规划质量 | 项目经理 | 制定质量标准 | 规划阶段 |
| 管理质量 | 质量保证部门 | 过程改进 | 执行阶段 |
| 控制质量 | 质量控制部门 | 检查可交付成果 | 执行阶段 |

### 13.4 建设团队 ⭐

**塔克曼阶梯理论（团队发展 5 阶段）：**
```
形成期 → 震荡期 → 规范期 → 成熟期 → 解散期
```

| 阶段 | 特点 | 管理策略 |
|------|------|----------|
| 形成期 | 相互认识，礼貌客气 | 明确目标、角色 |
| 震荡期 | 冲突增多，意见分歧 | 解决冲突、建立信任 |
| 规范期 | 建立规则，开始合作 | 强化规范、鼓励参与 |
| 成熟期 | 高效协作，自主工作 | 授权、支持 |
| 解散期 | 项目结束，团队解散 | 总结、庆祝 |

**培训类型：**
- 课堂培训、在线培训、在职培训
- 正式培训、非正式培训

### 13.5 管理团队 ⭐

**冲突管理 5 种策略：**

| 策略 | 特点 | 适用场景 |
|------|------|----------|
| 撤退/回避 | 暂时退出 | 问题不重要、需要冷静 |
| 缓和/包容 | 强调一致，淡化分歧 | 维持和谐 |
| 妥协/调解 | 各退一步 | 势均力敌、时间紧迫 |
| 强迫/命令 | 一方赢一方输 | 紧急情况、原则问题 |
| 合作/解决问题 | 双赢，直面问题 | 最重要、时间允许 ⭐ |

**激励理论：**

| 理论 | 提出者 | 核心内容 |
|------|--------|----------|
| 马斯洛需求层次 | 马斯洛 | 生理→安全→社交→尊重→自我实现 |
| 赫茨伯格双因素 | 赫茨伯格 | 保健因素（不满意）+ 激励因素（满意） |
| 麦格雷戈 XY 理论 | 麦格雷戈 | X 理论（消极）vs Y 理论（积极） |
| 期望理论 | 弗鲁姆 | 期望×工具性×效价 |

**输出：** 变更请求、团队绩效评估

## 第 15 章 项目风险管理 ⭐⭐⭐⭐⭐ **考试核心**

### 15.1 规划风险管理

**输出：** 风险管理计划

**风险分解结构 RBS：** 按风险类别分层展示

### 15.2 识别风险 ⭐

**工具与技术：**
- 头脑风暴、核对单分析
- 假设条件和制约因素分析
- SWOT 分析（优势、劣势、机会、威胁）
- 根本原因分析

**输出：** 风险登记册、风险报告

### 15.3 实施定性风险分析 ⭐

**评估维度：**
- 概率（高/中/低）
- 影响（高/中/低）
- 概率影响矩阵

**输出：** 风险登记册更新（优先级排序）

### 15.4 实施定量风险分析 ⭐

**工具与技术：**

| 工具 | 用途 |
|------|------|
| 蒙特卡洛模拟 | 模拟项目结果概率分布 |
| 敏感性分析（龙卷风图） | 识别高风险因素 |
| 决策树分析 | 计算预期货币价值 EMV |
| 影响图 | 展示变量关系 |

**预期货币价值 EMV：**
```
EMV = 概率 × 影响值
```

### 15.5 规划风险应对 ⭐⭐⭐⭐⭐ **案例分析必考**

#### 负面风险（威胁）应对策略：

| 策略 | 含义 | 示例 |
|------|------|------|
| 规避 | 消除威胁 | 改变计划、延长进度 |
| 转移 | 转给第三方 | 保险、外包、担保 |
| 减轻 | 降低概率或影响 | 冗余设计、测试 |
| 接受 | 不主动应对 | 建立应急储备 |

#### 正面风险（机会）应对策略：

| 策略 | 含义 | 示例 |
|------|------|------|
| 开拓 | 确保机会实现 | 分配优质资源 |
| 分享 | 与第三方合作 | 合资企业、团队 |
| 提高 | 增加概率或影响 | 增加资源、优化流程 |
| 接受 | 不主动应对 | 机会来了就利用 |

#### 应急应对策略：
- 预先定义的触发条件
- 触发时执行的应对措施

#### 输出：
- 变更请求
- 风险登记册更新（应对措施、责任人）

### 15.6 监督风险

- 跟踪已识别风险
- 识别新风险
- 评估风险过程有效性

## 第 16 章 项目采购管理 ⭐⭐⭐⭐

### 16.1 规划采购管理

**自制或外购分析：**
```
自制成本 vs 外购成本
考虑因素：核心能力、成本、风险、控制需求
```

**合同类型（必考）：**

| 合同类型 | 特点 | 风险承担 | 适用场景 |
|----------|------|----------|----------|
| **总价合同** | 固定价格 | 卖方风险高 | 范围明确 |
| 固定总价合同 FFP | 价格完全固定 | 卖方 | 范围非常明确 |
| 总价加激励费用 FPIF | 有激励条款 | 卖方为主 | 范围较明确 |
| 总价加经济价格调整 FP-EPA | 可调整（通胀） | 共担 | 长期合同 |
| **成本补偿合同** | 实报实销 + 费用 | 买方风险高 | 范围不明确 |
| 成本加固定费用 CPFF | 成本 + 固定费用 | 买方 | 研发项目 |
| 成本加激励费用 CPIF | 成本 + 激励费用 | 买方为主 | 有明确目标 |
| 成本加奖励费用 CPAF | 成本 + 主观奖励 | 买方 | 难以量化 |
| **工料合同 T&M** | 按时间/材料付费 | 共担 | 短期、应急 |

**记忆口诀：**
> **总价合同卖方险，范围明确好定价**
> **成本补偿买方险，范围不清用这个**
> **工料合同两边担，短期应急最适合**

### 16.2 实施采购

**过程：**
- 发布招标文件
- 投标人会议
- 提案评价
- 选择卖方

**输出：** 选定的卖方、协议

### 16.3 控制采购

- 管理采购关系
- 监督合同绩效
- 实施变更

**输出：** 采购关闭、变更请求

### 16.4 结束采购

- 完成单次项目采购
- 合同收尾

## ⚠️ 易错点提醒

### 1. 挣值管理计算
- **SV、CV 是减法，结果可正可负**
- **SPI、CPI 是除法，结果大于 1 是好的**
- **典型偏差用 CPI，非典型偏差直接减**
- **ETC 和 EAC 不要混淆**

### 2. 质量管理
- **管理质量 = 质量保证（过程导向）**
- **控制质量 = 质量检查（结果导向）**
- **质量成本中，一致性成本是预防，非一致性成本是失败**

### 3. 团队建设
- **震荡期冲突最多，不是成熟期**
- **合作/解决问题是最佳策略，不是妥协**

### 4. 风险管理
- **负面风险（威胁）和正面风险（机会）应对策略不同**
- **规避 vs 减轻：规避是消除，减轻是降低**
- **转移 vs 减轻：转移是给第三方，减轻是自己处理**

### 5. 合同管理
- **范围明确用总价合同，范围不清用成本补偿**
- **买方风险：成本补偿 > 工料 > 总价**
- **卖方风险：总价 > 工料 > 成本补偿**

### 6. 干系人管理
- **权力利益矩阵：高权力高利益是重点管理对象**
- **干系人参与程度：从抵制到支持需要策略**

## 📞 常用公式速查表

### 第 11 章 成本管理
```
三个基本参数：
PV = 计划工作量 × 预算单价
EV = 实际工作量 × 预算单价
AC = 实际工作量 × 实际单价

四个评价指标：
SV = EV - PV  （>0 提前，<0 落后）
CV = EV - AC  （>0 节约，<0 超支）
SPI = EV / PV （>1 提前，<1 落后）
CPI = EV / AC （>1 节约，<1 超支）

四个预测指标：
典型偏差：EAC = BAC / CPI
非典型偏差：EAC = AC + (BAC - EV)
ETC = EAC - AC
VAC = BAC - EAC

TCPI = (BAC - EV) / (BAC - AC)
```

### 第 14 章 沟通管理
```
沟通渠道数 = n(n-1)/2
```

### 第 15 章 风险管理
```
EMV = 概率 × 影响值
```

## 💡 案例分析答题技巧

### 挣值管理计算题步骤：
1. **先列出已知条件**（PV、EV、AC、BAC）
2. **计算 SV、CV、SPI、CPI**
3. **判断项目状态**（进度提前/落后，成本节约/超支）
4. **计算 EAC、ETC**（注意典型/非典型）
5. **给出建议**（如何改进）

### 风险应对策略选择题：
1. **判断是威胁还是机会**
2. **根据情景选择策略**
3. **注意关键词**：
   - "改变计划" → 规避
   - "买保险" → 转移
   - "增加测试" → 减轻
   - "不管它" → 接受

### 合同类型选择题：
1. **看范围是否明确**
2. **看风险承担偏好**
3. **看合同期限**
