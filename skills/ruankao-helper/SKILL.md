## 计算公式汇总

### 立项管理（第 7 章）
```
投资回收期 = 累计净现金流量开始出现正值的年份 - 1 + (上年累计净现金流量的绝对值 / 当年净现金流量)
NPV = Σ(净现金流量 / (1 + 折现率)^t)
ROI = (年平均利润 / 总投资) × 100%
```

### 进度管理（第 10 章）
```
三点估算：Te = (O + 4M + P) / 6
标准差：σ = (P - O) / 6
关键路径：网络图中最长路径
总时差：TF = LS - ES = LF - EF
自由时差：FF = min(紧后 ES) - 本活动 EF
```

### 成本管理（第 11 章）
```
PV = 计划工作量 × 预算单价
EV = 实际工作量 × 预算单价
AC = 实际工作量 × 实际单价

SV = EV - PV（>0 提前，<0 落后）
CV = EV - AC（>0 节约，<0 超支）
SPI = EV / PV（>1 提前，<1 落后）
CPI = EV / AC（>1 节约，<1 超支）

EAC = BAC / CPI（典型偏差）
EAC = AC + (BAC - EV)（非典型偏差）
ETC = EAC - AC
VAC = BAC - EAC
```

### 沟通管理（第 14 章）
```
沟通渠道数 = n(n-1)/2
```

### 风险管理（第 15 章）
```
EMV = 概率 × 影响值
```

## 记忆口诀库

### 五大过程组
> 启规执监收（启动、规划、执行、监控、收尾）

### 十大知识领域
> 整范进成质，资沟风采干
> （整合、范围、进度、成本、质量、资源、沟通、采购、干系人）

### 关键路径计算
> 正推取大求最早，逆推取小求最晚
> 总时差 = 晚 - 早，自由时差看紧后

### 变更控制流程
> 申请→评估→审批→实施→验证

### 三点估算
> 期望时间 = (乐观 + 4×最可能 + 悲观) ÷ 6
> 标准差 = (悲观 - 乐观) ÷ 6

### WBS 分解原则
> 100% 规则不能忘，工作包是最低层

### 质量老 7 工具
> 鱼骨找因流程图，核查收集数据忙
> 帕累托图抓重点，直方分布看清楚
> 控制图看稳不稳，散点图看相关性

### 团队发展 5 阶段
> 形震规成解（形成、震荡、规范、成熟、解散）

### 风险应对策略
> 负面：规避转移减轻接受

### 合同类型
> 总价合同卖方险，成本补偿买方险

### 专利保护期
> 发明二十年，实用十周年，外观十五年

## 易错点提醒

1. **范围确认 vs 质量控制**
   - 范围确认：客户验收（外部）
   - 质量控制：内部检查

2. **关键路径上的活动时差**
   - 总时差 = 0（不是自由时差）

3. **变更请求的审批**
   - CCB 审批，不是项目经理

4. **WBS 分解**
   - 工作包是最底层，不是活动

5. **三点估算公式**
   - Te = (O + 4M + P) / 6，不要记错系数

6. **IT 治理主体**
   - 董事会和管理层，不是 IT 部门

7. **项目组合 vs 项目集**
   - 项目组合：战略对齐
   - 项目集：收益协同

8. **项目 vs 运营**
   - 项目：临时性、独特性
   - 运营：持续性、重复性
