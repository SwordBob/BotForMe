# 🎯 挣值管理（EVM）公式速记卡片

## 📌 三个基本参数（必背！）

### 1. 计划价值（PV）
**含义**：计划完成工作的预算
**公式**：`PV = 计划工作量 × 预算单价`
**记忆**：**计**划的**价**值

### 2. 挣值（EV）
**含义**：实际完成工作的预算
**公式**：`EV = 实际工作量 × 预算单价`
**记忆**：**挣**得的**价**值

### 3. 实际成本（AC）
**含义**：实际完成工作的实际成本
**公式**：`AC = 实际工作量 × 实际单价`
**记忆**：**实**际**成**本

---

## 📊 四个评价指标（必背！）

### 1. 进度偏差（SV）
**公式**：`SV = EV - PV`
**判断**：
- `SV > 0`：进度**超前**（好）
- `SV < 0`：进度**落后**（不好）
- `SV = 0`：按计划

**记忆**：**挣**值减**计**划 = **进**度偏差

### 2. 成本偏差（CV）
**公式**：`CV = EV - AC`
**判断**：
- `CV > 0`：成本**节约**（好）
- `CV < 0`：成本**超支**（不好）
- `CV = 0`：按预算

**记忆**：**挣**值减**实**际 = **成**本偏差

### 3. 进度绩效指数（SPI）
**公式**：`SPI = EV / PV`
**判断**：
- `SPI > 1`：效率**高**（好）
- `SPI < 1`：效率**低**（不好）
- `SPI = 1`：按计划

**记忆**：**挣**值除以**计**划 = **进**度效率

### 4. 成本绩效指数（CPI）
**公式**：`CPI = EV / AC`
**判断**：
- `CPI > 1`：效率**高**（好）
- `CPI < 1`：效率**低**（不好）
- `CPI = 1`：按预算

**记忆**：**挣**值除以**实**际 = **成**本效率

---

## 🔮 四个预测指标（必背！）

### 1. 完工预算（BAC）
**含义**：项目总预算
**记忆**：**B**udget **A**t **C**ompletion

### 2. 完工尚需估算（ETC）
**含义**：完成剩余工作还需多少成本

#### 典型偏差（偏差会持续）
**公式**：`ETC = (BAC - EV) / CPI`
**记忆**：剩余工作（BAC-EV）除以当前效率（CPI）

#### 非典型偏差（偏差是特例）
**公式**：`ETC = BAC - EV`
**记忆**：剩余工作按原计划完成

### 3. 完工估算（EAC）
**含义**：预计项目总成本

#### 典型偏差
**公式1**：`EAC = AC + ETC`
**公式2**：`EAC = BAC / CPI`（常用）

#### 非典型偏差
**公式**：`EAC = AC + (BAC - EV)`

### 4. 完工偏差（VAC）
**公式**：`VAC = BAC - EAC`
**判断**：
- `VAC > 0`：预计**节约**
- `VAC < 0`：预计**超支**

---

## 📈 完工尚需绩效指数（TCPI）

### 含义
为了在预算内完成项目，剩余工作必须达到的成本绩效指数

### 公式1：按原预算（BAC）完成
```
TCPI = (BAC - EV) / (BAC - AC)
```

### 公式2：按新估算（EAC）完成
```
TCPI = (BAC - EV) / (EAC - AC)
```

### 判断
- `TCPI > 1`：需要**更高**效率
- `TCPI = 1`：按当前效率即可
- `TCPI < 1`：可以**降低**效率

---

## ⚡ 快速判断口诀

### 偏差判断
> **SV = EV - PV**：挣值减计划是进度
> **CV = EV - AC**：挣值减实际是成本
> **大于0是好，小于0是差**

### 指数判断
> **SPI = EV / PV**：挣值除以计划是进度效率
> **CPI = EV / AC**：挣值除以实际是成本效率
> **大于1是好，小于1是差**

### 典型/非典型记忆
> **典型偏差除CPI，非典型直接减**
> 题目出现"按当前趋势" → 典型
> 题目出现"特例"、"一次性" → 非典型

## 📝 常见易错点

### 易错点1：PV、EV、AC混淆
**区分**：
- PV：**应该**完成多少 → 计划
- EV：**实际**完成多少 → 挣得
- AC：**实际**花了多少 → 成本

### 易错点2：典型/非典型判断错误
**技巧**：
- 典型：题目说"按此趋势"、"绩效不变"
- 非典型：题目说"特例"、"一次性"、"已纠正"

### 易错点3：公式选择错误
**检查**：
- 典型偏差：公式中有CPI
- 非典型偏差：公式中无CPI

### 易错点4：计算顺序错误
**建议**：
- 先算基本参数（PV、EV、AC）
- 再算评价指标（SV、CV、SPI、CPI）
- 最后算预测指标（ETC、EAC等）

## 💡 记忆技巧

### 联想记忆法
- **PV**：Plan Value → 计划的价值
- **EV**：Earned Value → 挣得的价值
- **AC**：Actual Cost → 实际成本

### 对比记忆法
```
进度相关：SV = EV - PV, SPI = EV / PV
成本相关：CV = EV - AC, CPI = EV / AC
```

### 故事记忆法
> 项目经理**计划**（PV）完成10项工作，
> 实际**挣得**（EV）了8项工作的价值，
> **实际**花费（AC）了9项工作的成本。
> 
> 进度：8-10=-2（落后），8/10=0.8（效率低）
> 成本：8-9=-1（超支），8/9≈0.89（效率低）
