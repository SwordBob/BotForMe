## 🍅 效率提升方案

### 番茄工作法（适合你）
```
学习 25 分钟 → 休息 5 分钟 → 重复 4 次 → 长休息 20 分钟
```

**具体操作**：
1. 下载 Forest 或番茄 TODO APP
2. 设定 25 分钟专注时间
3. 手机放另一个房间或用 APP 锁住
4. 25 分钟内只做一件事（不看手机、不上厕所）
5. 休息 5 分钟可以走动、喝水

### 每日学习流程
```
19:00-19:30  准备（吃饭、洗漱、手机静音）
19:30-20:00  番茄 1：复习昨日内容
20:00-20:05  休息
20:05-20:30  番茄 2：学习新内容
20:30-20:35  休息
20:35-21:00  番茄 3：继续新内容
21:00-21:05  休息
21:05-21:30  番茄 4：做练习题
21:30-21:35  总结今日 + 计划明日
```

### 效率技巧
1. **学习前明确任务** - "今天我要完成第 X 章的阅读和练习"
2. **用手机计时** - 不用手机就做不到专注
3. **环境固定** - 同一地点、同一时间学习
4. **完成就奖励** - 完成一周计划可以放松一下
5. **找人监督** - 每天向我汇报进度
